# Security

如果你发现了与 `future of video` 相关的安全问题，请不要直接在公开 Issue 中披露细节。

建议做法：

- 使用私密渠道联系维护者
- 提供复现步骤、影响范围和风险判断
- 如果问题涉及密钥、鉴权、文件上传、任务执行或第三方模型调用，请明确说明

在收到报告后，维护者应优先处理以下类型问题：

- 认证绕过
- 任意文件读取 / 写入
- 路径穿越
- 未授权访问项目数据
- 凭据泄漏
- 命令执行
- 上传文件处理漏洞

如果你准备公开部署本项目，建议至少补充：

- HTTPS
- 限流
- 审计日志
- 生产级密钥管理
- 独立对象存储与权限隔离
- 上传文件扫描与内容安全策略
