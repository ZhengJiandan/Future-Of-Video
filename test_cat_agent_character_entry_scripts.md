# 猫咪特工专项测试剧本

这份文件用于专项测试这套逻辑：

- 新角色首次登场时是否会优先起新段
- 被标记的片段是否会额外预生成首帧
- 连续片段的上一段结尾是否尽量保留下一段继续出现的角色
- 后续视频阶段是否更容易通过上一段尾帧承接下一段，降低角色漂移

## 用例 1: 屋顶三角色接力登场

- `project_title`: 屋顶三角色接力追缉

  - `cat_char_lihuaqi`
  - `cat_char_juwan`

  - `cat_char_juwan` 首次登场应再起一段

## 使用建议

- 这两份剧本都适合直接粘贴到“完整剧本 -> 拆片”阶段。
- 用例 1 更适合测“连续追逐中多个新角色逐段登场”。
- 用例 2 更适合测“静态场景里多角色依次入场 + 尾帧角色交接”。
